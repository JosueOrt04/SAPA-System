# Sistema de Ventas SAPA

Sistema de ventas desarrollado en Java como solución a medida para una tienda
de electrónicos. La aplicación fue creada exclusivamente para un cliente,
adaptándose a sus necesidades específicas de operación y control comercial.

## Funcionalidades
- Gestión de productos
- Registro de ventas
- Control básico de inventario
- Administración de información comercial
- Flujo de ventas adaptado al negocio

## Tecnologías utilizadas
- Lenguaje: Java
- Base de datos: MySQL
- Arquitectura: Aplicación de escritorio
- Control de versiones: Git y GitHub

## Rol personal
Análisis de requerimientos del cliente, diseño y desarrollo completo de la
aplicación, enfocada en resolver un problema real de gestión de ventas.
